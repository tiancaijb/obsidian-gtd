# 0014 — Unit tests: i18n.ts

## 目标
为 `src/utils/i18n.ts` 添加测试，确保所有翻译键在中英文都存在。

## 背景
i18n 模块为插件提供中英双语支持。翻译遗漏会导致 UI 显示 key 名称而非翻译文本。需要确保所有在 `strings` 中定义的中文 key 都有对应的英文 key，反之亦然。

## 任务清单
- [ ] 测试 `t()`：中英文 key 都存在且返回非空字符串
- [ ] 测试 `t()`：不存在的 key 回退到英文然后返回 key 本身
- [ ] 测试 `groupTitles`：返回正确的分组标题数量（5 个）
- [ ] 测试 `metaKeywords`：中英文都有所有 7 个关键字
- [ ] 测试 `gtdFilenames`：所有 5 个文件条目都有值

- 翻译覆盖完整性验证
