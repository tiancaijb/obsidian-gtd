# 0009 — Unit tests: parser.ts

## 目标
为 `src/utils/parser.ts` 添加完整的单元测试。

## 背景
parser 是插件的核心模块，负责解析 Markdown 任务行、元数据行、序列化任务。所有任务数据流都经过它。没有测试的情况下，修改 parser 逻辑风险极高。

## 任务清单
- [ ] 测试 `isTaskLine`：正确识别含 `- ` 的行、拒绝普通文本行
- [ ] 测试 `isMetaLine`：正确识别缩进的元数据行
- [ ] 测试 `parseTaskLine`：
  - 标准任务 `- [ ] 写周报`
  - 已完成任务 `- [X] 买咖啡`
  - 带优先级 `- [ ] 重要任务  [#A]`
  - 带日期 `- [ ] 任务  PRIORITY: [#A]  SCHEDULED: <2026-06-28>`
  - 中文元数据 `- [ ] 任务  计划: <2026-06-28>`
  - 无效输入（空行、纯文本、列表项无 checkbox）
- [ ] 测试 `parseTaskLines`：多行元数据正确合并
- [ ] 测试 `serializeTask`：中英双语序列化结果正确
- [ ] 边界情况：元数据顺序变化、额外空格、特殊字符
